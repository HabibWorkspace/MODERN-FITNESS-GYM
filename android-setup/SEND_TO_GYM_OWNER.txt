================================================================================
ATTENDANCE SYNC - ANDROID DEVICE SETUP PACKAGE
================================================================================

Dear Gym Owner,

This package contains everything you need to set up automatic attendance 
syncing from your iFace950 device to your web app using an Android device.

================================================================================
WHAT'S INCLUDED:
================================================================================

1. FOR_GYM_OWNER.md          - Step-by-step setup guide (START HERE!)
2. android_device_sync.py    - The sync program
3. .env.example              - Configuration template
4. QUICK_START.md            - Quick reference guide
5. TROUBLESHOOTING.md        - Solutions to common problems
6. TRANSFER_INSTRUCTIONS.md  - How to copy files to Android

================================================================================
QUICK OVERVIEW:
================================================================================

WHAT YOU NEED:
- An Android phone or tablet (any old device works)
- WiFi connection (same network as your iFace950)
- USB cable (to transfer files)
- 15 minutes for setup

WHAT IT DOES:
- Automatically syncs attendance from iFace950 to your web app
- Runs 24/7 in the background
- Works even with screen off
- Starts automatically when device boots

COST:
- $0 if you have an old Android device
- Uses about 5W of power (very cheap to run)

================================================================================
SETUP STEPS (SUMMARY):
================================================================================

1. Install Termux app on Android (from F-Droid)
2. Install required software (copy-paste commands)
3. Transfer files to Android device
4. Configure settings
5. Start the sync
6. Done!

================================================================================
IMPORTANT NOTES:
================================================================================

⚠️ The commands in the guides are for the ANDROID DEVICE, not your computer!
⚠️ Install Termux from F-Droid, NOT from Google Play Store
⚠️ Keep the Android device plugged into power 24/7
⚠️ The device must stay on the same WiFi as your iFace950

================================================================================
WHERE TO START:
================================================================================

👉 Open "FOR_GYM_OWNER.md" and follow the steps!

It has pictures, detailed instructions, and everything you need.
